# Customer Shopping Behavior Analysis

## 📊 Project Overview
This project analyzes customer shopping behavior using 3,900 transactions to extract insights and improve business decisions.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI

## 🔍 Key Insights
- Customers using discounts still spend more
- Loyal customers generate higher revenue
- Certain products dominate sales

## 📈 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- SQL Business Queries
- Interactive Dashboard

## 📂 Project Structure
- data → dataset
- notebooks → python analysis
- sql → queries
- dashboard → Power BI file
- presentation → PPT

## 🚀 Outcome
Helps businesses improve marketing strategy, customer retention, and revenue growth.
